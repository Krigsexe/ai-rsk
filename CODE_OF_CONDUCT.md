# Code of Conduct

## Our Mission

ai-rsk exists to protect people's data. Every contributor shares this mission.

## Our Standards

**Expected behavior:**
- Be respectful and constructive
- Focus on facts, not opinions — back claims with CWE references, code examples, or documentation
- Welcome newcomers — security is hard, and people learning deserve patience
- Accept constructive criticism gracefully

**Unacceptable behavior:**
- Harassment, insults, or personal attacks
- Publishing others' private information
- Dismissing security concerns without evidence
- Trolling or deliberately derailing discussions

## Scope

This Code of Conduct applies to all project spaces: issues, pull requests, discussions, and any public space where you represent the project.

## Enforcement

Instances of unacceptable behavior may be reported to [julien.gelee@proton.me](mailto:julien.gelee@proton.me). All reports will be reviewed and investigated. The maintainer reserves the right to remove, edit, or reject contributions that violate this Code of Conduct.
